# 🃏 מחשבון פוקר - Poker Odds Calculator

אפליקציית ווב מותאמת מובייל לחישוב הסתברויות זכייה בפוקר בזמן אמת במהלך משחק.

## ✨ תכונות

- 🎴 **בחירת קלפים מהירה** - ממשק אינטואיטיבי עם כפתורים גדולים
- 🔒 **הסתרת קלפים** - אפשרות להסתיר את קלפי היד שלך (לפרטיות במשחק)
- 📊 **חישוב הסתברויות בזמן אמת** - באמצעות סימולציית Monte Carlo
- 🌙 **Dark Mode** - עיצוב אלגנטי בסגנון שולחן פוקר
- 📱 **מותאם מובייל** - כל התוכן בעמוד אחד ללא גלילה
- ⚡ **ביצועים מהירים** - JavaScript טהור ללא תלויות

## 🚀 איך להשתמש

### הרצה מקומית

פשוט פתח את הקובץ `index.html` בדפדפן:

```bash
# אופציה 1: פתיחה ישירה
open index.html

# אופציה 2: שרת מקומי (מומלץ)
python3 -m http.server 8000
# ואז פתח http://localhost:8000 בדפדפן
```

### העלאה ל-GitHub Pages

1. צור repository חדש ב-GitHub
2. העלה את כל הקבצים
3. הפעל GitHub Pages ב-Settings → Pages
4. בחר branch (main) ושמור
5. האפליקציה תהיה זמינה ב-`https://[username].github.io/[repo-name]`

### הוספה למסך הבית (PWA)

1. פתח את האפליקציה בדפדפן הטלפון
2. **Safari (iOS)**: לחץ על Share → Add to Home Screen
3. **Chrome (Android)**: לחץ על Menu → Add to Home Screen

## 🎮 מדריך שימוש

1. **בחר את הקלפים שלך** (2 קלפים)
   - לחץ על הקלפים מהרשת
   - אפשר להסתיר אותם באמצעות הכפתור "הסתר"

2. **בחר קלפים על השולחן**
   - Flop: 3 קלפים
   - Turn: 1 קלף (אופציונלי)
   - River: 1 קלף (אופציונלי)

3. **הגדר מספר שחקנים** במשחק (2-10)

4. **צפה בתוצאות**
   - אחוז זכייה
   - סוג היד הנוכחית
   - אפשרויות לשיפור (draws)

## 🧮 איך זה עובד?

האפליקציה משתמשת ב-**Monte Carlo simulation**:
- מריצה אלפי סימולציות של המשחק
- בוחנת את כל התוצאות האפשריות
- מחשבת אחוז זכייה מדויק על בסיס התוצאות

### דיוק החישוב

- **Pre-flop/Flop/Turn**: 5,000 סימולציות
- **River**: 1,000 סימולציות (פחות אי-ודאות)
- דיוק: ±1-2%

## 💡 טיפים לשימוש במשחק

- השתמש בכפתור "הסתר" כדי למנוע מאחרים לראות את הקלפים שלך
- עדכן את מספר השחקנים בהתאם למספר האנשים במשחק למדויק מקסימלי
- שים לב ל-draws - הם יכולים לשנות את ההחלטה שלך
- צבע ירוק = סיכוי גבוה (>70%), זהב = בינוני (40-70%), אדום = נמוך (<40%)
